# Feedback Skill

A skill for implementing in-app feedback forms.
First, check the application conditions in `../ui-skills.md` and `../frontend-design.md`.

---

## Trigger Phrases

- "Add a feedback feature"
- "I want to collect user feedback"
- "Create a bug report form"
- "Add a contact form"

---

## Features

- Feedback form
- Bug reporting
- Feature requests
- Satisfaction surveys

---

## Execution Flow

1. Check the application conditions for the constraint set
2. Review the project structure
3. Select the feedback type
4. Build the form UI
5. Create the API endpoint
6. Configure the data storage destination
